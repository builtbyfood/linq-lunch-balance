# Contributing

Thank you for helping improve this project!

## Reporting Issues

Please open an [Issue](../../issues) and include:
- What you expected to happen
- What actually happened (error message or node status)
- Your Node-RED version (`hamburger menu → Manage palette → About`)
- Your Home Assistant version

## Sharing Your School's GUID

The `LINQ_NUTRITION_URL` GUID varies by district. If you'd like to share yours to save others the DevTools step, open a [Discussion](../../discussions) with:
- District name and state
- Your GUID value

## Pull Requests

1. Fork the repo
2. Create a branch: `git checkout -b feature/my-improvement`
3. Make your changes
4. **Double-check that no personal data, tokens, or school-specific GUIDs are included**
5. Open a pull request with a description of what changed and why

## Security

If you discover a security issue, please open an Issue marked **[Security]** rather than a public pull request.
